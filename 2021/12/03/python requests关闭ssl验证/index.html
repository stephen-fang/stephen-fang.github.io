python requests关闭ssl验证

python requests关闭ssl验证

首先我们要看下是不是打开抓包工具，如果打开了抓包工具，那么请先关闭它，因为开启它之后运行python会自动改变端口，造成ssl错误。
直接在请求时关闭证书验证，即在请求的时候带上 verify =False，代码如下图:

Python3 提示warning 添加如下三种任意一种代码即可解决：

方式1：

import requests
requests.packages.urllib3.disable_warnings
方式2：

import warnings
warnings.filterwarnings(“ignore”)
方式3：

import urllib3
urllib3.disable_warnings(urllib3.exceptions.InsecureRequestWarning)
